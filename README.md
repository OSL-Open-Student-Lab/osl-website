# OSL Website API v.0.1
## Run application locally(For frontend devs)

```bash
$ git clone https://github.com/timaracov/osl.git

$ cd osl

$ python3 -m venv env

$ . env/bin/activate

$ pip install -r requirements.txt

$ export FLASK_APP=setup

$ export FLASK_ENV=development

$ flask run
```
